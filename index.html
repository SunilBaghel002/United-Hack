About United Hack

United Hack 2025 is a 36-hour online coding extravaganza that unites developers, designers, and innovators worldwide. Hosted on Discord and Zoom, this event challenges teams to create innovative web applications solving real-world problems. With a ₹20,000 prize pool and expert mentorship, United Hack is your platform to shine!

Innovate

Build cutting-edge web apps using any frontend tech to solve unique challenges.

Collaborate

Form teams of 2-4 to brainstorm, code, and learn from a global community.

Triumph

Compete for exciting prizes, recognition, and a chance to showcase your skills.

Prizes

Judging Criteria

Innovation

Eligibility Criteria

  • Open to students, professionals, and hobbyists worldwide.
  • Participants must register and pay the 200 INR team fee by June 11, 2025.
  • All team members must adhere to the code of conduct shared on Discord.

Frequently Asked Questions

Who can participate?

Anyone passionate about coding, aged 16+, can join. Teams must have 2-4 members.

What should we build?

Create a functional web application addressing a real-world problem. Use any frontend tech; mock data is allowed.

How do we submit our project?

Host your app on a live URL (e.g., Netlify, Vercel) and submit the link with your public GitHub repository by 6:00 PM IST on June 13, 2025.

Is the fee per person or per team?

The 200 INR fee is per team, not per person.

Can we use pre-existing code?

No, all code must be written during the hackathon. Open-source libraries or frameworks are allowed.

What if we face technical issues?

Join our Discord server for real-time support from mentors and organizers.
